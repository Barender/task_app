# TDCX Task

A full stack appliction build with React (vite) as frontend & Node-Express-Mongo as Backend.
Allows to create tasks on login, validations and authentication added on both ends to keep system secure robust.

# SETUP STEPS

- clone repo into local system.
- go to frontend directory and create .env file check .env.sample for more help.
- install all required packages.

```bash
yarn install
# or
npm install
```

- move to backend directory.
- create .env file and configure all variables check .env.sample for more help.
- install all packages

```bash
yarn install
# or
npm install
```

# RUN FRONTEND

- run only frontend (command for frontend directory)

```bash
yarn run dev
# or
npm run dev
```

# RUN BACKEND

- run only backend (command for backend directory)

```bash
yarn run server
# or
npm run sever
```

- run backend in --watch mode (command for backend directory)

```bash
yarn run backend
# or
npm run backend
```

- run frontend & backend concurrently (command fo backend directory)

```bash
yarn run dev
# or
npm run dev
```
